Instructions


    Python Libraries:

  1. Make sure you have a Python compiler set up and working
  2. Install the pygame library
  3. Install qiskit library
  4. Install python-dotenv library
  5. Install pytmx
  6. Click on QuantumOwl down below
  7. Download the project and run the main.py file

    Qiskit:

  1. Go to https://quantum-computing.ibm.com/
  2. Create an account with the Google Login
  3. In the middle of the screen, copy your API Token (this allows access to the quantum computing servers)
  4. Go to main.py
  5. Comment out Dontev
  6. Set TOKEN to your API Token
  7. Run main.py
