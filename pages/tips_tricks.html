Tips & Tricks

  For movement, to resolve the state you want, your best bet is to use another "H" gate on each state you would like to resolve to 0.
And a "y" gate, which rotates along the y-axis for each state you want to resolve to 1.

  For attack, do an x gate for the first line and an "i" gate for the line for a wing attack. Type two "i" gates for a hoot.
